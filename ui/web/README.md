# TG Mod Web UI

**TypeScript + HTML + CSS веб-интерфейс для TG Mod**

## 📦 Установка

```bash
cd ui/web
npm install
```

## 🚀 Разработка

```bash
npm run dev
```

Отворится на http://localhost:5173

## 🔨 Сборка

```bash
npm run build
```

Вывод в папке `dist/`

## 🎨 Дизайн

- **Цвета:** Тёмно-зелёный (#1f6f3b), Кремовый (#f7efe2), Охра/Бордовый (#8b5a3c)
- **Шрифты:** PT Sans, Rubik
- **Стиль:** Русский дизайн, тёмный режим по умолчанию
